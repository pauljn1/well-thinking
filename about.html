Projet Web - JUNIA ISEN 2026

Présentation du projet

Nous sommes une équipe de 9 étudiants en première année du cycle ingénieur JUNIA ISEN. Ce projet consiste à réaliser un site web utilisant nos compétences en programmation. Ce site web a pour but de réaliser un diaporama sous format d'arborescences avec possibilité de retour, dans un délai de 1 semaine.

Les besoins fonctionnels

Notre site web doit permettre à son utilisateur de créer des diaporamas avec la possibilité de les relier à plusieurs diapositives, pouvant créer lors de la lecture du diaporama un choix, ce qui rendrait l'utilisation des diaporamas plus interactive. Cela permet de créer des quiz ou bien des scénarios avec des choix intégrés pouvant ressembler à une histoire de type « Je suis le héros ».

L'édition de ce type de diaporama se fera sous format de drag and drop. Notre format de diaporama permettra de le personnaliser comme un diaporama standard et il y aura bien évidemment la fonctionnalité de pouvoir sauvegarder votre diaporama.

Les besoins techniques

Le développement repose sur les langages fondamentaux du web que sont le HTML5, le CSS3 et le JavaScript. Cette base technique permet de manipuler dynamiquement le DOM pour générer les diapositives et de structurer l'arborescence interactive via des fichiers JSON.

Afin de coordonner efficacement le travail de notre équipe de neuf personnes, nous utilisons Git et la plateforme GitHub pour centraliser le code source. Cet outil de versioning assure la synchronisation des développements en temps réel, permet de travailler simultanément sur l'éditeur et le lecteur, et garantit la sauvegarde permanente de chaque étape du projet.
